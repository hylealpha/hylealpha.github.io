# Hyle Alpha — Strategy Showcase

Public showcase of selected quantitative strategies, served via GitHub Pages at
[hylealpha.github.io](https://hylealpha.github.io/).

This repo is **content-only** — backtest data is generated by a private research
harness and pushed here by a publisher that strips internal labels, source code,
and any strategy not explicitly whitelisted for release.

## Structure

```
.
├── index.html          # Strategy list page
├── strategy.html       # Strategy detail page (reads ?id=...)
├── assets/
│   ├── css/style.css
│   └── js/{common,index,strategy}.js
└── data/
    ├── strategies.json # Registry consumed by index.html
    └── <strategy-id>/
        ├── meta.json
        ├── summary.json
        ├── nav.csv
        └── holdings.jsonl
```

## Local preview

No build step. Serve the directory with any static server:

```sh
cd hylealpha.github.io
python3 -m http.server 8000
# then open http://localhost:8000
```

## Adding a new strategy

The publisher in the private harness handles this end-to-end; this repo only
needs the resulting `data/<id>/` folder and a refreshed `data/strategies.json`.
